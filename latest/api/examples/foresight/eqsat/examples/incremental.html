final case class VersionMetadata[NodeT] extends Metadata[NodeT, VersionMetadata[NodeT]]

Metadata that tracks a global version number for the e-graph and a version number for each e-class. The global version number is incremented whenever new terms are added to the e-graph. Each e-class' version number is updated to the current global version number when a new term containing that e-class is added to the e-graph.

Metadata that tracks a global version number for the e-graph and a version number for each e-class. The global version number is incremented whenever new terms are added to the e-graph. Each e-class' version number is updated to the current global version number when a new term containing that e-class is added to the e-graph.

This implements the versioning scheme described in "Incremental Equality Saturation" by Soi, Driscoll, Wang, and Aiken. EGRAPHS 2025.
