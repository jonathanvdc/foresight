  1. abstract def apply[Repr <: EGraphLike[NodeT, Repr] with EGraph[NodeT]](egraph: Repr, reification: Map[Virtual, EClassCall], parallelize: ParallelMap): (Option[Repr], Map[Virtual, EClassCall])

    Applies the command to the given e-graph.

    Applies the command to the given e-graph.

    egraph

    The e-graph to which the command should be applied.

    reification

    A map from virtual e-class symbols to e-class calls that are used to reify the virtual e-class symbols.

    parallelize

    The parallelization strategy to use.

    returns

    The new e-graph, if it was changed, and a map from virtual e-class symbols to real e-class symbols. The map contains an entry for each virtual e-class symbol that is created by the command.

  2. abstract def definitions: Seq[Virtual]

    All e-class symbols that are defined by the command.

  3. abstract def simplify(egraph: EGraph[NodeT], partialReification: Map[Virtual, EClassCall]): (Command[NodeT], Map[Virtual, EClassCall])

    Simplifies the command for a given e-graph.

    Simplifies the command for a given e-graph. This method can be used to optimize the command for a specific e-graph. For example, a command that unions two e-classes that are already in the same e-class can be simplified to a no-op. Similarly, a command that adds an e-node that is already in the e-graph can be simplified to a reification map assignment.

    egraph

    The e-graph for which the command is to be simplified.

    partialReification

    A map from virtual e-class symbols to e-class calls that are used to reify the virtual e-clas symbols.

    returns

    The simplified command and a partial reification map for this command.

  4. abstract def uses: Seq[EClassSymbol]

    All e-class symbols that need to be reified to run the command.
