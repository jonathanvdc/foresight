MatchPriorities

trait MatchPriorities[NodeT, RuleT <: Rule[NodeT, MatchT, _], MatchT] extends AnyRef

A trait that defines a prioritization for matches found during rule application.

NodeT

The type of the nodes in the e-graph.

RuleT

The type of the rule.

MatchT

The type of the matches produced by the rule.
