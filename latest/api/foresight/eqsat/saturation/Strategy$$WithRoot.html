implicit final class WithRoot[NodeT, EGraphT <: EGraphLike[NodeT, EGraphT] with EGraph[NodeT], Data] extends AnyVal

An implicit class that adds operations to the Strategy trait for strategies that operate on an e-graph with a root.

NodeT

The type of the nodes in the e-graph.

EGraphT

The type of the e-graph that the strategy operates on, which must be a subtype of both EGraphLike and EGraph.

Data

The type of the data carried by the strategy.

    Creates a strategy that operates on an e-graph with a root.

    Creates a strategy that operates on an e-graph with a root. The root is typically used to extract trees from the e-graph. Roots are introduced at the beginning of the strategy and are closed at the end of the strategy.

    findRoot

    A function that finds the root of the e-graph.

  15. def thenRebase(extractor: Extractor[NodeT, EGraphT], areEquivalent: (Tree[NodeT], Tree[NodeT]) ⇒ Boolean = ...): Strategy[EGraphWithRoot[NodeT, EGraphT], (Data, Option[Tree[NodeT]])]

    Chains a rebasing operation to the strategy.

    Chains a rebasing operation to the strategy. The rebasing operation extracts a tree from the e-graph using the provided extractor and then rebases the e-graph with that tree. If the strategy does not change the e-graph, the rebasing operation is skipped. If the tree extracted from the e-graph is equivalent to the previously extracted tree, the rebasing operation is also skipped.

    extractor

    The extractor to use for extracting a tree from the e-graph.

    areEquivalent

    A function to check if two trees are equivalent. Defaults to structural equality.

    returns

    A new strategy that applies the rebasing operation after the original strategy.
