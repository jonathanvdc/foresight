final case class EGraphWithRecordedApplications[Node, Repr <: EGraphLike[Node, Repr] with EGraph[Node], Match <: PortableMatch[Node, Match]](egraph: Repr, applied: Map[String, Set[Match]]) extends EGraphLike[Node, EGraphWithRecordedApplications[Node, Repr, Match]] with EGraph[Node] with Product with Serializable

An e-graph that records the set of matches that have been applied to it.
