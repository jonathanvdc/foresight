RuleStats

foresight.eqsat.saturation.RuleStats
final case class RuleStats[NodeT, RuleT <: Rule[NodeT, MatchT, _], MatchT](rule: RuleT, matchLimit: Int, banLength: Int, bannedUntil: Option[Int], remainingMatches: Int)

Tracks dynamic backoff statistics for a single rule.

These stats are used to determine whether a rule is active, when it becomes eligible again, and how many more matches it may apply in the current activation window.

Value parameters

banLength

The number of iterations the rule will be banned after hitting its match limit.

bannedUntil

If banned, the iteration number at which the rule will become eligible again.

matchLimit

The total number of matches this rule can apply during its current active window.

remainingMatches

The number of matches this rule can still apply before triggering a ban.

rule

The rewrite rule being tracked.
