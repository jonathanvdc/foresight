final case class AnalysisMetadata[NodeT, A](analysis: Analysis[NodeT, A], results: Map[EClassRef, A]) extends Metadata[NodeT, AnalysisMetadata[NodeT, A]] with Product with Serializable

Analysis results for an e-graph's classes. This metadata can respond to changes in the e-graph.

NodeT

The type of the nodes in the e-graph.
