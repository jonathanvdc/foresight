final case class EGraphWithMetadata[NodeT, +Repr <: EGraphLike[NodeT, Repr] with EGraph[NodeT]] extends EGraphLike[NodeT, EGraphWithMetadata[NodeT, Repr]] with EGraph[NodeT] with Product with Serializable

An e-graph with associated metadata. Metadata is kept in sync with the e-graph upon every change.

NodeT
