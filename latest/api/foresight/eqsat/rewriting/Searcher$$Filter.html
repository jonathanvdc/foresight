Filter

Related Doc: package Searcher

final case class Filter[NodeT, MatchT, EGraphT <: EGraphLike[NodeT, EGraphT] with EGraph[NodeT]](searcher: Searcher[NodeT, Seq[MatchT], EGraphT], predicate: (MatchT, EGraphT) ⇒ Boolean) extends ReversibleSearcher[NodeT, MatchT, EGraphT] with Product with Serializable

A searcher that filters the output of another searcher based on a predicate.

NodeT

The type of the nodes in the e-graph.

MatchT

The type of the matches.

EGraphT

The type of the e-graph that the searcher searches in.

searcher

The searcher to filter.

predicate

The predicate to filter the output.
