  14. def requireIndependent(expr: Var[NodeT], slots: Slot*): Searcher[NodeT, Seq[PatternMatch[NodeT]], EGraphT]

    Requires that the expression bound to a variable is independent of a set of slots.

    Requires that the expression bound to a variable is independent of a set of slots. That is, the expression may not contain any of the slots in the set. Potential matches that do not satisfy this condition are filtered out.

    expr

    The variable to check.

    slots

    The slots to check.

    returns

    A searcher that filters out matches that do not satisfy the condition.
