PatternApplier

final case class PatternApplier[NodeT, EGraphT <: EGraphLike[NodeT, EGraphT] with EGraph[NodeT]](pattern: MixedTree[NodeT, Pattern[NodeT]]) extends ReversibleApplier[NodeT, PatternMatch[NodeT], EGraphT] with Product with Serializable

An applier that applies a pattern match to an e-graph.

NodeT

The type of the nodes in the e-graph.

EGraphT

The type of the e-graph that the applier applies the match to.

pattern

The pattern to apply.
