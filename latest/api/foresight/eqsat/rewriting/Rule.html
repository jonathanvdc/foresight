final case class Rule[NodeT, MatchT, EGraphT <: EGraphLike[NodeT, EGraphT] with EGraph[NodeT]](name: String, searcher: Searcher[NodeT, Seq[MatchT], EGraphT], applier: Applier[NodeT, MatchT, EGraphT]) extends Product with Serializable

A rule that can be applied to an e-graph.

NodeT

The type of the nodes in the e-graph.

MatchT

The type of the match.

EGraphT

The type of the e-graph that the rule can be applied to.

name

The name of the rule.

searcher

The searcher that finds matches of the rule in an e-graph.

applier

The applier that applies a match to an e-graph.
