trait SearcherLike[Node, Match, EGraphT <: ReadOnlyEGraph[Node], +This <: SearcherLike[Node, Match, EGraphT, This]]

A trait representing a searcher-like object that can be extended with continuation builders.

This trait now stores a continuation builder in CPS form: a function (Match => Boolean) => (Match => Boolean) which composes left-to-right (so filter.andThen(flatMap) runs filter then flatMap).

Type parameters

Match

The type of matches produced by the searcher-like object.

This

The concrete type of the searcher-like object, used for fluent method chaining.
